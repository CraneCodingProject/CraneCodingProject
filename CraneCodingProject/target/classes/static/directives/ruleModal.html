Language 

All exercise in  Crane Coding, we use JavaScript (JS) cause JavaScript  is a lightweight, interpreted, programming language with  first-class functions . Easy to learn and most well-known as the scripting language for Web pages, many non-browser environments also use it, such as node.js and Apache CouchDB. JS is a prototype-based, multi-paradigm, dynamic scripting language, supporting object-oriented, imperative, and declarative (e.g. functional programming) styles. Read more about JavaScript.( Based on : https://developer.mozilla.org ).

Rules

1.    With each exercise. We use function (read more about function here ) name Math and have one parameter is 'n' .It mean :

}

// Math ( 5 )  ==> output : 6.
